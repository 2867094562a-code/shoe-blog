# V10.7 评论删除与验证码

本版新增两个和评论区安全相关的功能。

## 1. 后台评论删除

后台新增“评论”栏目：

- 查看所有文章评论
- 按昵称、邮箱、评论内容、文章标题搜索
- 跳转查看对应文章
- 删除垃圾评论 / 广告评论

删除接口需要后台登录，不会暴露给前台用户。

## 2. 前台评论验证码

文章页评论表单新增验证码：

- 访客评论前必须填写简单数学验证码
- 验证码 10 分钟过期
- 提交失败后自动刷新验证码
- 用后端签名 token 校验，避免只靠前端判断

## 3. 部署注意

更新到这个版本后，需要重新上传代码到 GitHub，然后 Render 重新部署。

Render 建议继续使用：

```bash
Build Command: npm install -g pnpm@9.12.3 && pnpm install --no-frozen-lockfile
Start Command: node src/server.js
```

如果浏览器显示旧页面，按 `Ctrl + F5` 强制刷新。
