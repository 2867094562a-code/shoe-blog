# v10.5 遗漏与优化清单

这版重点不是继续堆功能，而是把后台体验和后续可维护性补齐。

## 已补齐

1. 站点设置右侧加入主页实时预览
   - Logo、站点标题、页眉导航、Hero、首页卡片、作者信息、分类标签、文章示例会同步变化。
   - 小屏幕自动改为上下布局，避免表单被压窄。

2. 设置页排版优化
   - 左侧专心编辑，右侧预览固定在可视区域。
   - 预览区内部模拟前台首页，而不是普通文字预览。

3. 预览性能优化
   - 使用 100ms 防抖，输入时不会每个字都暴力重绘。
   - 预览不请求前台接口，直接读取当前表单内容。

4. 兜底与空态
   - 没有文章时会显示文章卡片示例。
   - 没有分类标签时会显示默认示例。
   - 没有 Logo 时显示默认渐变 Logo。

## 建议下一步再做

1. 图片上传前自动压缩为 WebP
   - 可以明显节省 Supabase Storage / R2 空间。

2. 自定义 CSS 安全白名单
   - 现在适合个人使用；如果以后开放给别人注册，需要限制危险 CSS。

3. 视频平台模板化
   - 后台可以提供 B 站 / 抖音 / 西瓜 / YouTube 的填空模板。

4. 模块数据从 Markdown 迁移到 JSON
   - 现在兼容 Markdown，迁移成本低；如果后续模块越来越多，建议把文章模块独立存 JSON。

5. 后台文件拆分
   - admin.js 已经偏大，后续建议拆成：editor.js、settings.js、preview.js、upload.js、taxonomy.js。

6. SEO 设置
   - 页面 / 文章可增加 SEO 标题、描述、封面图、是否允许索引。

7. 数据导出 / 导入
   - 后台增加一键导出 JSON 备份，方便换服务器和防止误删。
