# V10.13 最终自查与修复说明

这版是收尾正式版，重点做稳定性、同步逻辑、移动端、视频、评论和部署配置修复。

## 已修复

1. 视频默认暂停
   - 清理第三方视频链接里的 autoplay=1 / autoPlay=1 / auto_play=1。
   - B站 / YouTube 链接自动补 autoplay=0。
   - iframe allow 权限移除 autoplay。
   - 后台实时预览与前台渲染保持一致。

2. 评论体验
   - 验证码错误返回 400，不再误显示为服务器 500。
   - 评论加入简单频率限制，减少刷屏。
   - 后台评论管理可删除评论。

3. Clean URL 冲突
   - 新建文章和新建页面时，slug 会互相检查，避免文章和页面占用同一个地址。
   - 避免使用 api / admin / archive / category / tag / search / uploads / css / js 等系统保留路径。

4. 站点设置同步
   - 设置保存逻辑支持不存在的 key 自动补写，避免新字段后台保存后前台拿不到。
   - 前台 / 后台 API 请求使用 no-store，减少缓存导致的不同步。

5. Render 部署
   - package.json 固定 Node 20.18.1。
   - render.yaml 改为 pnpm 构建，避免 npm Exit handler never called。
   - 删除 package-lock.json，避免误走 npm ci。
   - .gitignore 追加 package-lock.json / pnpm-lock.yaml，减少错误提交。

6. 系统检查
   - 后台新增“系统检查”页。
   - 可查看 Node 版本、运行环境、数据库模式、图片存储方式、Supabase 是否配置、图片桶、后台入口、上传大小限制。
   - 不展示密钥明文。
