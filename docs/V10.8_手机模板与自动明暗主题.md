# V10.8 手机模板与自动明暗主题

## 新增内容

1. 服务端 UA 识别
   - 手机 UA 会优先返回 `public/mobile.html`。
   - 电脑 UA 继续返回 `public/index.html`。
   - 可用 `?view=mobile` 强制手机模板，用 `?view=desktop` 强制桌面模板，方便测试。

2. 手机端独立模板
   - 新增 `public/mobile.html`。
   - 手机端使用独立顶部栏、横向导航、紧凑 Hero、作者信息条、分类标签卡片和更适合阅读的内容流。
   - 手机端关闭鼠标跟随光斑和卡片倾斜，避免移动端卡顿和误触。

3. 自动跟随系统明暗模式
   - 第一次访问会读取设备的 `prefers-color-scheme`。
   - 设备是深色模式时，自动使用 Night 主题。
   - 设备是浅色模式时，使用后台设置的默认主题色。
   - 用户手动点击主题按钮后，会记住用户选择，不再强制跟随系统。

## 更新部署注意

Render 继续使用：

```bash
Build Command:
npm install -g pnpm@9.12.3 && pnpm install --no-frozen-lockfile
```

```bash
Start Command:
node src/server.js
```

必须保留 `.node-version`：

```text
20.18.1
```

## 测试方法

电脑上测试手机模板：

```text
https://你的域名/?view=mobile
```

电脑上强制桌面模板：

```text
https://你的域名/?view=desktop
```

手机直接访问会自动使用手机模板。
