# V10.6 设置同步与模块开关

## 解决的问题

1. 后台站点设置保存后，前台部分模块没有同步显示。
2. 某些模块不是每个站都需要，应该允许关闭展示。
3. 主题色可能被浏览器本地缓存覆盖，管理员误以为后台设置不生效。

## 改动

- `module_visibility` 新增到后端设置字段。
- 后台站点设置新增“模块显示开关”。
- 前台 `loadSite()` 统一读取设置并执行显示/隐藏。
- 前台 API 请求加 `cache: no-store`。
- 后台保存设置前统一同步隐藏字段：导航、卡片、分类标签、模块开关。
- 后台主页实时预览也跟随模块开关变化。

## 模块开关说明

关闭模块不会删除数据，只会隐藏展示。
