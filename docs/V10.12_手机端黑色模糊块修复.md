# V10.12 手机端黑色模糊块修复

## 修复内容

手机端主卡片下方在非夜间主题下出现大块黑色高斯模糊的问题已修复。

原因是手机模板继承了桌面端的大尺寸背景光斑和 Hero 内部光斑，部分手机浏览器在浅色主题下会把混合背景 / blur / 透明层渲染成黑色块。

## 处理方式

- 手机端禁用桌面大尺寸背景光斑 `.bg-orb`
- 手机端禁用 Hero 内部大光斑 `.hero-glow`
- 增加轻量圆形氛围光 `.mobile-safe-glow`
- 限制 Hero 区域 `overflow` 和 `contain: paint`
- 降低手机端噪点层透明度

## 更新后测试

手机端打开：

```text
https://你的域名/?view=mobile
```

如果浏览器缓存旧样式，清缓存或无痕模式打开。
